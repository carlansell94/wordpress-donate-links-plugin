Add a donate block to your site. Includes styles for Buy Me a Coffee, Ko-Fi, Patreon and PayPal.

== Description ==

This plugin is provides a block to display donate links on your site.

Has style support for Buy Me a Coffee, Ko-Fi, Patreon and PayPal, though as links are not validated, any link can be added under each setting.

Can also display a message above the links, and the site logo (configurable in the cusomiser).

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/donate-links` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
